# input

规定了用户可以在其中输入数据的输入字段。输入字段可以表现为多种组件，取决于type属性。

## 属性

##### 主要

<table>
<tr>
	<th>autofocus</th>
	<td>bool</td>
	<td>规定输入字段在页面加载时是否获得焦点</td>
</tr>
<tr>
	<th>disabled</th>
	<td>bool</td>
	<td>当input元素加载时禁用此元素</td>
</tr>
<tr>
	<th>form</th>
	<td>id</td>
	<td>规定输入字段所属的一个或多个表单</td>
</tr>
<tr>
	<th>name</th>
	<td>string</td>
	<td>用于表示表单提交时的参数名</td>
</tr>
<tr>
	<th>readonly</th>
	<td>bool</td>
	<td>规定输入字段为只读</td>
</tr>
<tr>
	<th>type</th>
	<td>enum</td>
	<td>规定input元素的类型</td>
</tr>
</table>

##### 可选

<table>
<tr>
	<th>accept</th>
	<td>mime</td>
	<td>规定通过文件上传来提交的文件的类型</td>
	<td>file</td>
</tr>
<tr>
	<th>alt</th>
	<td>string</td>
	<td>图像输入的替代文本</td>
	<td>image</td>
</tr>
<tr>
	<th>autocomplete</th>
	<td>enum</td>
	<td>规定是否使用输入字段的自动完成功能
	<br/>on / off</td>
	<td>text、search、url、tel、email、password、date、datetime、datetime-local、month、time、week、range、color</td>
</tr>
<tr>
	<th>checked</th>
	<td>bool</td>
	<td>规定此input元素首次加载时应当被选中</td>
	<td>checkbox、radio</td>
</tr>
<tr>
	<th>formaction</th>
	<td>url</td>
	<td>覆盖表单的action属性</td>
	<td>image、submit</td>
</tr>
<tr>
	<th>formenctype</th>
	<td>mime</td>
	<td>覆盖表单的enctype属性
	<br/>application/x-www-form-urlencoded：（默认）在发送前编码所有字符
	<br/>multipart/form-data：不对字符编码
	<br/>text/plain：空格转换为 "+" 加号，但不对特殊字符编码</td>
	<td>image、submit</td>
</tr>
<tr>
	<th>formmethod</th>
	<td>enum</td>
	<td>覆盖表单的method属性
	<br/>post / get</td>
	<td>image、submit</td>
</tr>
<tr>
	<th>formnovalidate</th>
	<td>bool</td>
	<td>覆盖表单的novalidate属性。如果使用该属性，则提交表单时不进行验证</td>
	<td>text、search、url、tel、email、password、date、datetime、datetime-local、month、time、week、range、color</td>
</tr>
<tr>
	<th>formtarget</th>
	<td>target</td>
	<td>规定如何打开action url，覆盖表单的target属性，参见<a href="../参考/target.md">提交目标</a></td>
	<td>image、submit</td>
</tr>
<tr>
	<th>height</th>
	<td>number<br/>percent</td>
	<td>定义高度</td>
	<td>image</td>
</tr>
<tr>
	<th>list</th>
	<td>id</td>
	<td>引用包含输入字段的预定义选项的datalist的id</td>

</tr>
<tr>
	<th>max</th>

	<td>number、range、date、datetime、datetime-local、month、time、week</td>
</tr>
<tr>
	<th>maxlength</th>
	<td>int</td>
	<td>输入字段中的字符的最大长度</td>

</tr>
<tr>
	<th>min</th>
	<td>int<br/>date</td>

</tr>
<tr>
	<th>required</th>
	<td>bool</td>
	<td>指示输入字段的值是必需的</td>

</tr>
<tr>
	<th>size</th>
	<td>int</td>
