# select

select元素可创建单选或多选菜单。

## 约束

对称标签。

只能包含option标签。

## 属性

##### 可选

<table>
    <tr>
		<th>autofocus</th>
		<td>bool</td>
		<td>在页面加载时是否获得焦点</td>
	</tr>
	<tr>
		<th>disabled</th>
		<td>bool</td>
		<td>规定禁用该下拉列表</td>
	</tr>
	<tr>
		<th>form</th>
		<td>id</td>
		<td>所属的一个或多个表单</td>
	</tr>
	<tr>
		<th>multiple</th>
		<td>bool</td>
		<td>规定可选择多个选项</td>
	</tr>
	<tr>
		<th>name</th>
		<td>string</td>
		<td>下拉列表的名称</td>
	</tr>
	<tr>
		<th>required</th>
		<td>bool</td>
		<td>规定文本区域是必填的</td>
	</tr>
	<tr>
		<th>size</th>
		<td>int</td>
		<td>下拉列表中可见选项的数目</td>
	</tr>
</table>
