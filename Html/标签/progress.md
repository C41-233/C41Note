# progress

定义运行中的任务进度。

## 约束

对称标签。

## 属性

##### 可选

<table>
<tr>
    <th>max</th>
    <td>number</td>
    <td>规定需要完成的值</td>
</tr>
<tr>
    <th>value</th>
    <td>number</td>
    <td>规定进度的当前值</td>
</tr>
</table>

## 默认样式

```
progress {
    display: inline-block;
    box-sizing: border-box;
}
```
