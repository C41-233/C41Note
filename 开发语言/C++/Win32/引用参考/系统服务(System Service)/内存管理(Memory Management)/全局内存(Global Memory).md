# 全局内存

##### 函数
- GlobalAlloc 分配全局内存
- GlobalFree 释放全局内存
- GlobalHandle 获取虚拟地址对应的全局内存句柄
- GlobalLock 锁定全局内存
- GlobalMemoryStatus 获取当前内存状态的动态信息
- GlobalMemoryStatusEx 获取当前内存状态的动态信息
- GlobalReAlloc 重新分配全局内存
- GlobalSize 获取全局内存大小
- GlobalUnlock 解除锁定全局内存

##### 结构体
- MEMORYSTATUS 内存状态
- MEMORYSTATUSEX 内存状态
