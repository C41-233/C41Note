# typeof

typeof是一个运算符，用于获取操作数的类型。

```
typeof operand
typeof(operand)
```

| 结果 | 类型 |
|---|---|
| "undefined" | undefined |
| "boolean" | boolean | 
| "string" | string |
| "number" | number |
| "bigint" | BigInt |
| "symbol" | Symbol |
| "function" | Function |
| "object" | 其他任何对象，包括object、null、Boolean、Number、String |
