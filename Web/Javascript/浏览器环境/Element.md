# Element

代表文档的所有元素的通用接口。

## 继承

Object / [EventTarget](EventTarget.md) / [Node](Node.md) / Element

| 名称 | 类型 | 描述 |  writable | enumerable | configurable |

| attributes | NamedNodeMap | 所有属性的集合 | - | - | - |
| childElementCount | number | 子元素结点的数量 | - | - | - |
| children | HTMLCollection | 所有子元素结点的集合 | - | - | - |
