# Event

##### 拖拽事件

| 事件 | 描述 |
|---|---|
| drag | 当拖动元素时 |
| dragend | 当拖动操作结束时 |
| dragenter | 当元素被拖动至有效的拖放目标时 |
| dragleave | 当元素离开有效拖放目标时 |
| dragover | 当元素被拖动至有效拖放目标上方时 |
| dragstart | 当拖动操作开始时 |
| drop | 被拖动元素正在被拖放时 |
