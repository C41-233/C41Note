# HTML Entity

在HTML中，某些字符是预留的。需要用到`<`、`>`、`空格`等符号时，会错误的把它们与标记混在一起，非常不利于编码。

如果希望正确地显示预留字符，我们必须在HTML源代码中使用字符实体。

以名称描述实体时，格式为`&<name>;`，以编码描述实体时，格式为`&#<number>;`。例如，`<`表示为`&lt;`或者`&#60;`。

实体列表：https://www.freeformatter.com/html-entities.html

| 字符 | 实体名称 | 实体编号 |
|---|---|---|
| " | quot | 34 |
| ' | apos | 39 |
| & | amp | 38 |
| <  | lt | 60 |
| >	 | gt | 62 |
| 空格 | nbsp | 160 |
