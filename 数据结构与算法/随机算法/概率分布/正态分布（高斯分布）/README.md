# 正态分布（高斯分布）

## 定义
若随机变量X的概率密度为![](https://latex.codecogs.com/gif.latex?f(x)=\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}(x\in&nbsp;R,\sigma>0))，称X服从参数为μ、σ的正态分布，记![](https://latex.codecogs.com/gif.latex?N(\mu,\sigma^2))。
