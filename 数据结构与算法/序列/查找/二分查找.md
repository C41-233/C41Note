# 二分查找

二分查找法又叫折半查找法，在有序数组中查找，时间复杂度为O(logn)。

## 元素搜索

### 实现

##### Java
``` Java
/*
 * 要求数组array有序，返回值value的下标，如果没有找到，则返回-1。
 */
public static <T> int BinarySearch(T[] array, T value)
{
    int left = 0; 
    int right = array.length-1;
    while(left<=right){
        int mid = (left+right)/2;
        if(array[mid]==value){
            return mid;
        }
        else if(array[mid]<value){
            left = mid+1;
        }
        else{
            right = mid-1;
        }
    }
    /*
     * 此时：
     * left的值是value插入时所在的坐标，即array[left]是第一个大于该value的值。当value>array[length-1]时，该值等于length。
     * right的值等于left-1，即array[right]是第一个小于value的值。当value<array[0]时，该值等于-1。
     */
    return -1;
}
```
