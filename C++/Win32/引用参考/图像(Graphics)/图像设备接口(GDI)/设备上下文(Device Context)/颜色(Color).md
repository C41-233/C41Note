# 颜色
##### 函数
- GetNearestColor 获取与指定混色最接近的纯色
- GetSysColor 获取系统颜色
- SetSysColors 设置系统颜色

##### 宏函数
- GetBValue 获取蓝色分量
- GetGValue 获取绿色分量
- GetRValue 获取红色分量
- RGB 生成32位RGB颜色值

##### 消息
- WM_SYSCOLORCHANGE 系统颜色改变时

##### 数据类型
| type | far pointer | 位数 | 符号 |
|---|---|---|---|
| COLORREF | LPCOLORREF | 32 | unsigned |
