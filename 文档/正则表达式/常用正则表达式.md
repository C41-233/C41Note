# 常用正则表达式

##### IPv4地址
```
^(([01]?[0-9]{1,2})|(2[0-4][0-9])|(25[0-5]))(\.(([01]?[0-9]{1,2})|(2[0-4][0-9])|(25[0-5]))){3}$
```

##### Email
```
^([a-z0-9A-Z]+[-|\.]?)+[a-z0-9A-Z]@([a-z0-9A-Z]+(-[a-z0-9A-Z]+)?\.)+[a-zA-Z]{2,}$
```

##### 偶数个0与奇数个1构成的字符串
```
^(0(00|11)*(01|10)|1)(00|11|(10|01)(00|11)*(01|10))*$
```
