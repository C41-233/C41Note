# IMUL
有符号乘法，分为8位乘法与16位乘法。

##### 8位乘法
- 操作数1：AL
- 操作数2：R或(address)
- 积：AX

##### 16位乘法
- 操作数1：AX
- 操作数2：R或(address)
- 积：DX高16位，AX低16位

## 指令格式
```
IMUL R/M
```
