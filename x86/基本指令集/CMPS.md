# CMPS<width>

根据<width>指定的位宽不同，分别进行串比较，将(DS:SI)中的值与(ES:DI)中的值比较，影响标志位。

若DF=0，则SI:DI递增；若DF=1，则SI:DI递减。

# 指令格式
```
CMPS8
CMPS16
CMPS32
```
