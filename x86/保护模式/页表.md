# 页表

## 页表项

第一级页表，占4KB，每项4B，共1024项。每项指向一个第二级页表。

第二级页表，每个占4KB，每项4B，每个有1024项。每项指向一个物理页。

### 标志位
| 位 | 描述 |
|---|---|
| [31] | 指向的页基址或页表基址 |
| [11] | 由程序员自定义 |
| [8] | （G）指示全局页。G=1表示全局页，G=0表示非全局页 |
| [7] | （PAT/PS） |
| [6] | （D）是否被写入。1表示已被写入，0表示未被写入。页或页表刚被加载到物理内存中时被内存管理程序清零，处理器在第一次写入此页或页面时设置，且不会自动清除 |
| [5] | （A）是否被访问。1表示已被访问，0表示未被访问。页或页表刚被加载到物理内存中时被内存管理程序清零，处理器在第一次访问此页或页面时设置，且不会自动清除 |
| [4] | （PCD）控制对单个页或页表的缓冲。PCD=0可以缓冲，PCD=1不可以缓冲。当CR0的CD=PCD1时被忽 |
| [3] | （PWT）控制对单个页或页表的缓冲策略。PWT=0使用write-back策略，PWT=1使用write-through策略。当CR0的CD=PCD1时被忽略 |
| [2] | （U/S）特权级，1表示用户级，0表示系统级 |
| [1] | （R/W）读写权限，0表示只读，1表示读写 |
| [0] | （P）存在位，表示当前条目所指向的页或页表是否在物理内存中。P=0表示不在内存中，如果处理器试图访问此页，将会产生异常；P=1表示在内存中 |
