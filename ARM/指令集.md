# ARM指令集

ARM指令的通用格式为：

```
opcode{cond}{S} Rd, Rn {, operand2}
```

- opcode：指令
- cond：条件选项
- S：开启影响CPSR寄存器选项
- Rd：目标寄存器
- Rn：第一操作数寄存器
- oprand2：第二操作数
