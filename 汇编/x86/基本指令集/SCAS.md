# SCAS<width>
按字节（8位）串检索。

8位下，将AL中的值减去(ES:DI)中的值，影响标志位。若DF=0，则SI:DI递增；若DF=1，则SI:DI递减。

16位下，将AX中的值减去(ES:DI)中的值，影响标志位。。若DF=0，则SI:DI递增；若DF=1，则SI:DI递减。

32位下，将EAX中的值减去(ES:EDI)中的值，影响标志位。。若DF=0，则SI:EDI递增；若DF=1，则SI:EDI递减。

<width>指定位宽。

## 指令格式
```
SCAS<width>
```
