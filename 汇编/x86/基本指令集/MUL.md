# MUL

无符号乘法，分为8位乘法与16位乘法。

##### 8位乘法
- 操作数1：8位，AL
- 操作数2：8位，R或address
- 积：16位，AX

##### 16位乘法
- 操作数1：16位，AX
- 操作数2：16位，R或address
- 积：32位，DX高16位，AX低16位

## 指令格式
```
MUL r/m
```
