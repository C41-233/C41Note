# DAA

压缩BCD码调整。

AL为两个压缩BCD数相加后的结果，如果AL的低半字节大于9或AF=1，则AL增加06H，置AF为1；然后，如果AL高半字节大于9或CF=1，则AL加上60H，置CF为1。

如果两个条件都不满足，则置AF和CF为0。

## 指令格式
```
DAA
```
