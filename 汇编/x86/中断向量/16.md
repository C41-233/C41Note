# INT 16H

由BIOS提供的键盘中断。AH指定服务编号。

## 00H
从键盘缓冲区读一个输入，并将其从缓冲区删除。

如果缓冲区为空，则会阻塞，直到一个键被按下。

#### 输出
- AH 扫描码（1字节）

| 位 | 描述 |
|---|---|
| [7] | insert，是否开启删除 |
| [6] | CapsLock，是否开启大写 |
| [5] | NumLock，是否开启小键盘数字 |
| [4] | ScrollLock，是否指示灯亮 |
| [3] | Alt，是否按下 |
| [2] | Ctrl，是否按下 |
| [1] | 左shift，是否按下
| [0] | 右shift，是否按下 |

- AL 字符ASCII码
